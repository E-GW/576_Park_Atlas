# GEOG 576 Midterm Project Repo - 
project description:
For your Midterm Project, you are asked to submit your idea for a web application using Esri.js. The application should have a responsive front-end design for mobile and desktop web browsers and a backend based on ArcGIS Online. The full stack should allow users to query, write, and update the database. You aim to provide an intuitive user interface that allows users to interact with the data easily.
